## Overview

Docsify turns one or more Markdown files into a Website, with no build process required.

### Features

- No statically built html files
- Simple and lightweight
- Smart full-text search plugin
- Multiple themes
- Useful plugin API
- Emoji support


**Shipped version:** 4.4.4~ynh3
